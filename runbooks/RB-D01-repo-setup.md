# Runbook: D01 - GitHub Repository Setup

## Purpose
Set up initial GitHub repository for IT lab portfolio with proper structure and documentation.

## Scope
Applies to new portfolio repositories.

## Prerequisites
- Git installed
- GitHub account
- Internet connection

## Tools
- Git (CLI)
- Command Prompt
- GitHub Web UI
- Notepad

---

## Procedure

### Step 1: Create Repository on GitHub
1. Go to GitHub
2. Click New Repository
3. Name: it-support-lab-portfolio
4. Set to Public
5. Create without README

---

### Step 2: Clone Repository to Local Machine

```bat
mkdir C:\Projects
cd C:\Projects
git clone <repo_url>
cd it-support-lab-portfolio
