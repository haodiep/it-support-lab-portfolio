# IT Support & System Administration Lab Portfolio (30-Day Plan)

## Overview
This repository documents a 30-day hands-on IT support and system administration learning plan.

The project focuses on building real-world skills in:
- Virtualization
- Networking
- Windows Server administration
- Troubleshooting
- Documentation
- Automation (PowerShell)

All labs are documented with evidence, runbooks, and knowledge base articles.

---

## Environment

- Host OS: Windows 10/11
- Hypervisor: Oracle VirtualBox
- Main VMs:
  - DC01 (Windows Server – Domain Controller)
  - Linux VMs (Ubuntu)
  - Windows Client (future)

Network Design:
- NAT Network: Internet access
- Host-only: Internal lab network

## Troubleshooting Methodology

All incidents follow this structure:

1. Identify symptoms
2. Isolate root cause
3. Apply fix
4. Validate result
5. Document in runbook

This simulates real IT support workflows.
